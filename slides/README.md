# Talk slides — Agentic UI with Angular Dynamic Components

Slidev deck for the `ng-a2ui` talk. Self-contained — does not touch the Angular app's dependencies.

```bash
cd slides
npm install
npm run dev      # serves the deck at http://localhost:3030
npm run export   # optional: export a PDF (slides-export.pdf)
```

Edit `slides.md`. Press `f` for fullscreen, `o` for slide overview, `p` for presenter/notes mode.
